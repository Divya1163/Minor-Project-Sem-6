🚀 QUICK START

  1. Install Dependencies:
     $ npm install

  2. Start Development Server:
     $ npm run dev

  3. Open in Browser:
     http://localhost:3000

  4. Start Using:
     • Enter thermal parameters or select a sample scenario
     • Click "Predict Comfort Level"
     • View prediction results with confidence scores
     • Click "Analytics" tab to explore visualizations

  5. Deploy to Production:
     $ vercel --prod
