## 🚀 Getting Started

### 1. Installation (1 minute)
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

### 2. Make Your First Prediction
- Enter thermal parameters or select a sample scenario
- Click "Predict Comfort Level"
- View the prediction result with confidence score

### 3. Explore Analytics
- Click the "Analytics" tab
- View comprehensive visualizations
- Analyze patterns and model performance

### 4. Deploy (< 5 minutes)
```bash
vercel --prod
```
