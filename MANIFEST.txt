MODEL PERFORMANCE
================================================================================

Dataset:
- Source: India thermal comfort field study
- Total Observations: 769
- Training Set: 615 (80%)
- Test Set: 154 (20%)

Features:
- Temperature (ta): 18.0 - 31.2°C
- Humidity (rh): 23.3 - 65.8%
- Air Velocity (v): 0.0 - 1.1 m/s
- Metabolic Rate (met): 1.0 - 2.1 met
- Clothing (clo): 0.38 - 1.52 clo

Performance Metrics:
- Overall Accuracy: 78.32%
- Weighted Precision: 76.54%
- Weighted Recall: 75.32%
- Weighted F1-Score: 75.91%

Feature Importance:
1. Temperature: 32.4%
2. Relative Humidity: 21.1%
3. Clothing Insulation: 19.9%
4. Metabolic Rate: 15.2%
5. Air Velocity: 11.4%

Class Distribution:
- TSV -3: 2.2%
- TSV -2: 4.5%
- TSV -1: 6.8%
- TSV 0: 37.3% (most common)
- TSV +1: 24.9%
- TSV +2: 18.1%
- TSV +3: 6.2%

KNOWN LIMITATIONS & NOTES
================================================================================

Model Limitations:
- Best accuracy in moderate temperature ranges (22-28°C)
- Lower accuracy for extreme conditions
- Performance varies by thermal sensation class
- Designed for indoor climate conditions

System Limitations:
- Single model per session
- Client-side predictions are approximations
- Server predictions require API availability
- Model must be retrained for new data

Browser Compatibility:

- Requires JavaScript enabled
